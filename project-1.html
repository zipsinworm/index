CNN Clasification
*GE
9개 Category를 갖는 초음파 이미지 CNN 모델 Training.
Training 한 모델이 C++에서 동작하기 위한 방법구현과 테스트를 통한 ResNet50 등
(하단 이미지) 모델을 이용하여 실시간 분류. Configuration에서 각 Category에 Accuracy 설정 가능 하며, 실제 Scan에서의 Accuracy가 설정된 값을 넘게 되면 해당 매핑된 Category로 자동 또는 수동으로 변경.
GPU가 아닌 CPU로만 사용 가능한지 System Performance 테스트 진행.
▶ 개발 환경(Ubuntu 18.04.6 LTS / Docker - Python 3.8.5, Tensor 2.5.0) 구축 /
모델 Training(ResNet50, 101, 152 - V1, V2 & MobileNet V2) / System Integration 및 Engineering 테스트 진행하였습니다.
