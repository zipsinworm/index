Automation Test Tool
*Siemens
랜덤 테스트(Monkey Test)의 일종으로, 장비의 Front Panel과 더치스크린 버튼을 랜덤으로

또한, Log를 읽어 Playback 가능하게 하여 재현하기 힘든 이슈, 현장에서 발생한 다양한
이슈 등 디버깅에 활용. Script를 이용하여 시나리오 반복 테스트 추가.
▶ Feature 리딩 / 사용자 Interface 정의 / Software Architecture 설계 및 구현(C++, C#, Python) / System Integration 및 Engineering Test 진행 / 문서 작성하였습니다.

Middleware Tool
*Digital Craft.INC
Cannon 복합기 Middleware 개발, Windows 제어 소프트웨어를 Linux(SUSE)로 포팅하여
Linux에서도 System 제어 및 해당 레지스터값을 가져올 수 있도록 개발.
Fuse를 이용하여 File system에서 Cache를 사용, 동일 반복 작업 시, Disk 읽기 속도를 1.5x 배 이상 퍼포먼스 향상.
Database의 구조 및 데이터를 Linux 내에서 CUI로 간단하게 표현 가능 하도록 하여 개발 초기 시간을 줄임.
▶ 사용자 워크플로우 정의 / 구현(C) / Engineering 테스트 진행 / 문서 작성하였습니다.
